# 🐾 PetCare App

Aplicación PWA para control y seguimiento de salud de mascotas.

## 🚀 Cómo subir a Netlify

### Opción A — Netlify Drop (más fácil, sin cuenta GitHub)

1. Abre tu terminal en esta carpeta
2. Ejecuta:
   ```bash
   npm install
   npm run build
   ```
3. Ve a **https://app.netlify.com/drop**
4. Arrastra la carpeta **`dist`** que se generó
5. ¡Listo! Tu app estará online en segundos 🎉

---

### Opción B — GitHub + Netlify (recomendada, con auto-deploy)

1. Sube esta carpeta a un repositorio en GitHub
2. Ve a **https://app.netlify.com** → "Add new site" → "Import from Git"
3. Selecciona tu repositorio
4. Netlify detecta automáticamente la configuración del `netlify.toml`
5. Haz clic en **Deploy** ✅

Cada `git push` actualizará tu sitio automáticamente.

---
